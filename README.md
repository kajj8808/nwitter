# Nwitter
Cloning Twitter(mini) with React and Frirebase

## Git Page 
- https://kajj8808.github.io/nwitter

## 개발에 사용한 것
- React 
- Firebase(backend 대용)

### React 
- Reset Css - styled-reset
- Styled - styled-components
- @fortawesome (https://fontawesome.com/)
- Hooks 

### Firebase
- auth(이메일 , 소셜 인증)
- firestore(문서방식 DB Service)
- storage(이미지 저장용)
